User Friendly Specification of Interactive Robotic Applications in Service Settings

Polimi has developed a framework for developing robotic applications in service settings (such as hospitals, home assistance, education) that can handle the unpredictability of human behavior. The starting point of the framework is a formal model of the interactive scenario. On the other hand, the framework's target users are experts in logistics and planning but not formal modeling.
This thesis project aims to develop a user-friendly Domain-Specific Language (DSL) to make the framework accessible to its users. The DSL should allow users to specify (mainly, though not exclusively): the floor layout, the starting configuration of robots, how many humans are involved, their physical characteristics, and what service they are requesting.
The thesis outcome should also include the automated model-to-model (DSL to formal model) transformation process and the validation on significant use cases.

Formalizing a Cognitive Model of the Human Decision-Making Process

Polimi has developed a framework for developing robotic applications in service settings (such as hospitals, home assistance, education) that can handle the unpredictability of human behavior. The framework features a formal model of humans with a probabilistic formulation of free will, as they may ignore the robot's instructions or make autonomous decisions altogether.
This thesis project aims at developing a more sophisticated formulation of the human decision-making process based on cognitive models existing in the literature (preferably with probabilistic features).
The result should be integrated into the existing formal model and validated to assess the improvement compared to the current model via formal verification of significant use cases.

Verification and Deployment of Multi-Robot Multi-Human Interaction Scenarios

Polimi has developed a framework for developing robotic applications in service settings (such as hospitals, home assistance, education) that can handle the unpredictability of human behavior. The framework is currently limited to scenarios featuring multiple humans interacting with a single robot. The dynamism of selected application fields, though, calls for more complex solutions involving multi-robot teams.
This thesis aims at extending the current formal model with robot-robot interaction patterns, such as two robots exchanging tasks or carrying out a task jointly, and developing a more complex robot controller architecture that can handle multi-robot scenarios.
The thesis outcome should also cover the conversion of the developed models into executable code to validate the enriched framework on significant use cases.
